atto_cloze
=================

This is an editor plugin to provide an interface for creating and
modifying embedded answer (cloze) questions. To install add to the Atto
editor plugins directory and visit notifications to update database. Then
add 'cloze' to the list of buttons in Atto toolbar settings. The button
should appear in the editor only while editing questions. Select question
type and a form will appear form entering answers.
